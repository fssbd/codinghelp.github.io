Avro Keyboard

Avro Keyboard (Bengali: অভ্র কিবোর্ড) is a free and open source graphical keyboard software developed by OmicronLab for the Microsoft Windows, Ubuntu and as well as other flavours of Linux and MacOS operating system. It is the first free Unicode and ANSI compliant Bengali keyboard interface for Windows that was published on 26 March 2003.

Notepad++

Notepad++ is a text editor and source code editor for use with Microsoft Windows. It supports tabbed editing, which allows working with multiple open files in a single window. The project's name comes from the C increment operator.
